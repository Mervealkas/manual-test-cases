# manual-test-cases
# Manuel Test Senaryoları

Bu repository, örnek bir web uygulaması için hazırlanmış manuel test senaryoları ve hata (bug) raporu örneklerini içermektedir.

## Kapsam
Bu projede aşağıdaki fonksiyonlar test edilmiştir:
- Kullanıcı giriş (Login) işlemleri
- Kullanıcı kayıt (Register) işlemleri
- Zorunlu alan kontrolleri
- Hata mesajları ve kullanıcı uyarıları


## İçerik
- Login_Test_Cases.xlsx  
  → Giriş ekranı için hazırlanmış test senaryoları
- Register_Test_Cases.xlsx  
  → Kayıt ekranı için hazırlanmış test senaryoları
- Bug_Report_Sample.xlsx  
  → Tespit edilen bir hataya ait örnek bug raporu

## Amaç
Bu çalışma, yazılım test süreçleri konusundaki temel bilgi seviyemi ve manuel test case ile bug raporu hazırlayabilme yetkinliğimi göstermek amacıyla oluşturulmuştur.
